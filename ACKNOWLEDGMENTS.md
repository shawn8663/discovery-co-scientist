# Acknowledgments

Discovery Co-Scientist builds on ideas and implementation work from several
open scientific-agent efforts.

- The original open-source Co-Scientist project from Kaimen Inc. provided the
  starting repository structure and the first implementation base for this
  branch.
- Gottweis et al., "Accelerating scientific discovery with AI co-scientists,"
  inspired the general multi-agent hypothesis architecture: Supervisor,
  Generation, Reflection, Ranking, Evolution, Proximity, and Meta-review.
- Ghareeb et al., "A multi-agent system for automating scientific discovery,"
  inspired the Robin-style lab-in-the-loop therapeutic discovery workflow:
  assay selection, therapeutic candidate generation, experimental analysis,
  and experiment-informed regeneration.
- FutureHouse Robin, Crow, Falcon, and Finch informed the dedicated
  therapeutic discovery prompt stack and controlled analysis-agent design.

This repository is independent and is not affiliated with Google, FutureHouse,
Kaimen Inc., or the referenced paper authors.
